About this event

What are Android Study Jams?

Android Study Jams are community-organized study groups for people to learn how to build Android apps in the Kotlin programming language, using curriculum provided by Google.

During the course, you will get a general overview of mobile development for Android, learn basic programming concepts as well as core vocabulary and concepts in Android, get hands-on experience building a small app using the Kotlin language
